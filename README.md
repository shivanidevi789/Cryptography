# 🔐 Cryptography Programs in Python

This repository contains a collection of Python programs that demonstrate fundamental concepts of cryptography, encryption, hashing, number theory, and secure communication. These implementations were created for learning and academic purposes.

---

## 📖 Overview

Cryptography is the practice of securing information through encryption, decryption, hashing, and mathematical algorithms. This project includes implementations of classical ciphers, modern encryption techniques, RSA key generation, modular arithmetic, and hashing algorithms.

---

## 📋 Programs Included

### 1️⃣ Caesar Cipher

Implementation of the Caesar Cipher for encryption and decryption of plaintext using a fixed shift value.

**Concepts Covered:**

- Symmetric Encryption
- Character Shifting
- Encryption & Decryption

---

### 2️⃣ Substitution Cipher

A program that encrypts and decrypts text by replacing each alphabet letter with another predefined letter.

**Concepts Covered:**

- Monoalphabetic Substitution
- Encryption Mapping
- Decryption Mapping

---

### 3️⃣ Transposition Cipher

Implementation of a transposition cipher that rearranges the positions of letters in the plaintext.

**Concepts Covered:**

- Permutation Techniques
- Rearrangement of Characters
- Classical Cryptography

---

### 4️⃣ Basic DES Encryption & Decryption

A simplified implementation of the Data Encryption Standard (DES) algorithm.

**Concepts Covered:**

- Block Cipher
- Symmetric Key Encryption
- Feistel Structure

---

### 5️⃣ Triple DES (3DES)

Implementation demonstrating Triple DES encryption and decryption for enhanced security.

**Concepts Covered:**

- Triple Encryption
- Symmetric Cryptography
- DES Enhancement

---

### 6️⃣ Block Cipher Mode (CBC)

Implementation of Cipher Block Chaining (CBC) mode for block cipher operations.

**Concepts Covered:**

- CBC Mode
- Initialization Vector (IV)
- Block Chaining

---

### 7️⃣ Modular Arithmetic Calculator

A calculator that performs arithmetic operations under modular arithmetic.

**Concepts Covered:**

- Modular Addition
- Modular Subtraction
- Modular Multiplication
- Modular Division

---

### 8️⃣ RSA Key Generation

A program to generate RSA public and private keys using two prime numbers.

**Concepts Covered:**

- Public Key Cryptography
- Prime Numbers
- Key Generation
- Euler's Totient Function

---

### 9️⃣ Euclid's Algorithm for GCD

Implementation of Euclid’s Algorithm to calculate the Greatest Common Divisor (GCD) of two numbers.

**Concepts Covered:**

- Number Theory
- Recursion
- Mathematical Algorithms

---

### 🔟 MD5 Hash Generator

A program that computes the MD5 hash value of a given string.

**Concepts Covered:**

- Cryptographic Hash Functions
- Data Integrity
- Message Digests
